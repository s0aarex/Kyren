Como usar

GUIA RÁPIDO
  1. Abra o Hydra Launcher.
  2. Acesse a seção Importar / Fontes.
  3. Cole o link da source Kyren (arquivo JSON do repositório).
  4. Confirme a importação e aguarde a sincronização.

Após a importação, os jogos disponíveis na source Kyren passam a aparecer automaticamente no launcher.

DICAS IMPORTANTES
  • Se algum item não aparecer, aguarde alguns minutos e atualize a lista.
  • Links quebrados ou erros pontuais podem ocorrer — reporte para correção.
  • Mantenha sempre a versão mais recente da source.
